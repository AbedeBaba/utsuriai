# UtsuriAI - AI Fashion Photography Platform

AI-powered fashion photography platform that transforms product photos into stunning model images.

## About

UtsuriAI helps fashion brands create professional model photos using AI technology. Upload your product images and generate high-quality fashion photography in minutes.

## Features

- AI-generated fashion models
- Multiple poses and backgrounds
- Support for various clothing categories
- Template-based generation system
- Multi-language support (English & Turkish)

## Technologies

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase
